### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn (for this i'm gonna use npm)

### Installation

1. Clone the repository:
```bash
git clone <https://github.com/MyCannedFood/Portofolio.git>
cd Portofolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
